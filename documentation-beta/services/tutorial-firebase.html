findOne function

FirebaseService.findOne
source

Retrieve a resource from Firebase server.


FirebaseService.findOne( params, [[success],[error]] )

Retrieve a single resource from the Firebase server.

Parameters

params {Object}
Values to filter the request or results with. params.path
function (data)

Returns

data
A callback on successful retrieval. The callback receives the retrieved resource as data
data.error
A callback when an error occurs. The callback receives an object data.error

Use

FirebaseService.findOne() is used to retrieve a model instance from the server.

Use findOne like:

{% highlight html %} FirebaseService.findOne({ path: 'posts/post_one' }, function(data) { if(!data.error) { console.log(data); } else { console.log(data); } }); {% endhighlight %}

The return data should look like:

{% highlight html %} {author: "John Deauthor", body: "Conveniently harness fictionalized communities high-impact markets under whelm go.", title: "The post title ", node_id: "post_one"} {% endhighlight %}

saveData function

FirebaseService.saveData
source

Save/update to Firebase server.


FirebaseService.saveData( params, [[error]] )

Retrieve a single resource from the Firebase server.

Parameters

params {Object}
Values to filter the request or results with. params.path , params.data
function (data)

Returns

data.error
A callback when an error occurs. The callback receives an object data.error

Using saveData to create an instance.

FirebaseService.saveData() is used to retrieve a model instance from the server.

If saveData() is called an instance is created in Firebase.

{% highlight html %} FirebaseService.saveData({ path: '/author/posts', data: ({author:"John Deauthor", body: "post body content.", title: "Another post title "}) }, function(data) { if(!data.error) { console.log(data); } else { console.log(data); } }); {% endhighlight %}

If updateData() is update the instance created on Firebase base on the node key in this case . -KT-kz68IJrOajjwg865

{% highlight html %} FirebaseService.updateData({ path: '/author/posts/-KT-kz68IJrOajjwg865', data: ({author:"John Deauthor", body: "post body content.", title: "Another post title "}) }, function(data) { if(!data.error) { console.log(data); } else { console.log(data); } }); {% endhighlight %}
